# 🏯 NDL古文書自動翻刻ツール - 完全ガイド

国立国会図書館（NDL）の「新編武蔵風土記稿」を Claude Vision を使って自動翻刻するシステムです。

## 📋 提供されるファイル

| ファイル | 説明 | 使用方法 |
|---------|------|--------|
| `transcribe_ndl.py` | Python スクリプト版 | ローカル実行 |
| `transcriber_app.md` | ブラウザ版アプリ | Claude.ai で実行 |
| `README.md` | 詳細ドキュメント | 参照 |

---

## 🚀 最速スタートガイド

### 方法1: ブラウザで今すぐ試す（推奨）

1. `transcriber_app.md` ファイルを開く
2. React コードをコピー
3. Claude.ai に新しいチャットで貼り付け
4. 「このページを翻刻」ボタンをクリック

**メリット**: インストール不要、すぐに開始可能
**デメリット**: 一度に複数ページを処理しづらい

---

### 方法2: ローカル環境で実行

#### セットアップ（初回のみ）

```bash
# 1. Anthropic パッケージをインストール
pip install anthropic

# 2. API キーを設定
export ANTHROPIC_API_KEY="sk-ant-..."
```

#### 使用方法

```bash
# 最初の5ページを翻刻
python3 transcribe_ndl.py

# カスタム範囲を翻刻
python3 transcribe_ndl.py 1 10      # ページ 1〜10
python3 transcribe_ndl.py 50 55     # ページ 50〜55
```

**メリット**: 大量ページ処理に適している、カスタマイズ容易
**デメリット**: インストール・設定が必要

## 📚 翻刻ツールの機能

### 翻刻対象資料

- **タイトル**: 新編武蔵風土記稿
- **巻**: 巻之１８～２４（豊島郡・葛飾郡）
- **出版社**: 内務省地理局
- **出版年**: 明治17年（1884年）
- **ページ数**: 114ページ
- **ライセンス**: パブリックドメイン（自由利用可）

### 翻刻の処理内容

| 処理項目 | 内容 |
|--------|------|
| **古字体の統一** | 旧漢字や異体字を現代字体に変換 |
| **変体仮名の解釈** | ゐ→い、ゑ→え など旧仮名遣いを現代遣いに |
| **句点の挿入** | 元のテキストに適切な句読点を追加 |
| **レイアウト保持** | 元の行構成・段落構造をできるだけ保持 |
| **不明文字の記録** | 読めない文字は【？】で記載 |
| **注釈の追加** | 必要な注釈は【】内に記す |

## ⚠️ 重要な注意事項

### 翻刻精度について

- ✅ 通常の江戸時代の活字資料は 85〜95% の精度で翻刻可能
- ⚠️ 画質が悪い場合、手書き部分がある場合は精度が低下
- 🔴 **学術的引用や重要な利用では、必ず原本と照合してください**

### API 使用上の注意

- Claude API のクレジットが消費されます（有料）
- 大量ページ処理時はレート制限に注意
- 不正な用途での利用は禁止

### ライセンスと引用

本資料はパブリックドメインですが、利用時には以下を記載してください：

```
『新編武蔵風土記稿』巻之１８～２４
内務省地理局編、明治17年 (1884年)
国立国会図書館デジタルコレクション
https://dl.ndl.go.jp/info:ndljp/pid/763978
```

## 🤝 よくある質問

### Q. インストールなしで使える？
**A.** はい。ブラウザ版（`transcriber_app.md`）をご利用ください。

### Q. すべてのページを一度に処理できる？
**A.** ローカルスクリプト版なら可能です：
```bash
python3 transcribe_ndl.py 1 114
```

### Q. 別の NDL 資料も翻刻できる？
**A.** はい。スクリプトを編集して別資料に対応可能です。詳細は `README.md` を参照。

### Q. 翻刻結果を編集・修正したい場合は？
**A.** ダウンロードしたテキストファイルを任意のエディタで編集できます。

### Q. オフライン環境で使える？
**A.** いいえ。Claude API への接続が必須です。
